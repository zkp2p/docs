## Summary

- Updates Step 12 (Configure Order Limits) in the seller guide to explain how default minimum and maximum order sizes are now set automatically
- Deposits at or below 101 USDC default the minimum to ~5% of the deposit; deposits above 101 USDC default the minimum to 101 USDC
- Describes how to override defaults via the Order Limits section

## Why

The create-deposit flow shipped a new default minimum order behavior (zkp2p-clients #701 / #705). The existing guide only showed a static example of 5 USDC and did not explain the automatic defaults, leaving makers unaware of why their minimum order differs from the old percentage-based formula.

Closes zkp2p/docs#77
